# 🛍️ Munki – E-commerce de Kits Imprimibles

**Munki** es una tienda online desarrollada como e-commerce para la venta de kits imprimibles digitales. Los usuarios pueden navegar por el catálogo, seleccionar productos, y realizar compras con métodos de pago locales e internacionales. El sitio entrega los archivos por vía digital, eliminando la necesidad de logística física.

## ⚙️ Funcionalidades principales

- Visualización de productos y detalles individuales  
- Carrito de compras
- Formulario de contacto

## 🛠️ Tecnologías utilizadas
- HTML
- CSS
- JavaScript
